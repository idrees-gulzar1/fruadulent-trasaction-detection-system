ABOUT

Dataset Information

Based on the Kaggle PaySim Financial Transactions Dataset, which contains over 6.3 million transactions.

Features include transaction amount, type (PAYMENT, TRANSFER, etc.), sender and recipient balance deltas, and timestamp data.

ML Models

  • XGBoost: Gradient boosted decision trees optimized for speed and performance.
  • LightGBM: Fast, distributed, high-performance gradient boosting framework.
  • Random Forest: Ensemble learning method using multiple decision trees.
  • Logistic Regression: Statistical model used for binary classification.
